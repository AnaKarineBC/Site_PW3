    Aqui você aprenderá a estilizar e criar designs profissionais para suas páginas web. Nosso conteúdo abrange:

    • Introdução à CSS3 - Conheça a linguagem de estilização moderna
    • Formatação de Documentos - Aprenda a linkar CSS no HTML e estilizar elementos

    Conteúdo Programático:

    CSS3 (Cascading Style Sheets) é a linguagem padrão para estilizar e formatar páginas web. Define cores, fontes, espaçamento, layout e muitos outros aspectos visuais.

    Características principais:

    • Linguagem de estilo (não é programação)
    • Controla a aparência visual das páginas
    • Suporta animações e transições
    • Responsividade (adaptação para diferentes telas)
    • Seletores avançados e media queries

    Como funciona:

    seletor {
  propriedade: valor;
  propriedade2: valor2;
}

    Formas de linkar CSS no HTML:

    1. CSS Externo (Recomendado):

    <link rel="stylesheet" href="estilos.css">

    2. CSS Interno:

    <style>
  h1 { color: blue; }
</style>

    3. CSS Inline:

    <h1 style="color: blue;">Título</h1>

    Seletores CSS principais:

    Seletor Descrição Exemplo
    Elemento Seleciona por tag p { color: red; }
    Classe Seleciona por classe .destaque { color: blue; }
    ID Seleciona por ID único #titulo { color: green; }
    Atributo Seleciona por atributo [type="text"] { border: 1px solid; }
    Pseudo-classe Estados especiais a:hover { color: orange; }

    Propriedades CSS comuns:

    • color - Cor do texto
    • background-color - Cor de fundo
    • font-size - Tamanho da fonte
    • font-family - Tipo de fonte
    • margin - Espaço externo
    • padding - Espaço interno
    • border - Borda do elemento
    • width/height - Largura e altura
