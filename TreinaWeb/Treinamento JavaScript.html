    Treinamento em JavaScript

    Este curso vai te ensinar a criar sites e aplicativos usando uma das mais modernas linguagens de programação

    Bem-vindo ao Treinamento de JavaScript!

    Aqui você aprenderá a adicionar interatividade e lógica às suas páginas web. Nosso conteúdo abrange:

    • Introdução à JavaScript - Conheça a linguagem de programação para web
    • Fundamentos do JavaScript - Entenda como funciona e veja exemplos práticos
    • Programação Funcional - Aprenda a usar JavaScript para dar vida ao HTML

    Conteúdo Programático:

    JavaScript é a linguagem de programação padrão para criar interatividade em páginas web. É executada no navegador do usuário.

    Características principais:

    • Linguagem de programação (diferente de HTML e CSS)
    • Executa no navegador do cliente (Client-side)
    • Permite interatividade e animações
    • Manipula elementos HTML dinamicamente
    • Trata eventos de usuário (cliques, digitação, etc)

    Exemplo básico:

    <script>
  console.log("Olá, Mundo!");
  let nome = "João";
  alert("Bem-vindo, " + nome);
</script>

    Onde usar JavaScript:

    • Validar formulários
    • Criar efeitos visuais
    • Manipular o DOM (Document Object Model)
    • Requisições AJAX
    • Desenvolvimento de aplicações web complexas

    Tipos de dados básicos:

    Tipo Descrição Exemplo
    String Texto "Olá" ou 'Mundo'
    Number Números inteiros e decimais 42 ou 3.14
    Boolean Verdadeiro ou Falso true ou false
    Array Lista de valores [1, 2, 3]
    Object Coleção de propriedades {nome: "João", idade: 30}

    Variáveis:

    let variavel = "valor";  // Escopo local
const constante = 10;     // Não pode ser alterada
var antigo = "obsoleto";  // Escopo global (evitar)

    Operadores principais:

    • Aritméticos: +, -, *, /, %
    • Comparação: ==, ===, !=, <, >, <=, >=
    • Lógicos: &&, ||, !
    • Atribuição: =, +=, -=, *=, /=

    Estruturas de controle:

    // Condicional
if (condicao) {
  // faça algo
} else {
  // faça outra coisa
}

// Loop
for (let i = 0; i < 10; i++) {
  console.log(i);
}

    Funções: blocos de código reutilizáveis que realizam tarefas específicas.

    // Declaração
function saudacao(nome) {
  return "Olá, " + nome;
}

// Função flecha (Arrow Function)
const soma = (a, b) => a + b;

// Chamando funções
console.log(saudacao("Maria"));
console.log(soma(5, 3));

    Manipulando o DOM:

    // Selecionar elementos
let elemento = document.getElementById("id");
let classe = document.querySelector(".classe");

// Alterar conteúdo
elemento.textContent = "Novo texto";
elemento.innerHTML = "<strong>Negrito</strong>";

// Alterar estilos
elemento.style.color = "blue";
elemento.classList.add("ativa");

    Tratando eventos:

    // Evento de clique
botao.addEventListener("click", function() {
  alert("Botão clicado!");
});

// Eventos comuns
// - click: clique do mouse
// - submit: envio de formulário
// - change: mudança em input
// - mouseover: mouse sobre elemento
// - keydown: tecla pressionada

    Dicas importantes:

    • Use console.log() para debugar
    • Valide sempre os dados do usuário
    • Use const por padrão, let quando necessário
    • Evite variáveis globais
    • Comentários descrevem o propósito do código
