    Conteúdo Programático:

    HTML5 (HyperText Markup Language) é a versão mais recente da linguagem de marcação padrão para criar páginas web. Ela fornece a estrutura e o conteúdo das páginas na internet.

    Características principais:

    • Linguagem de marcação (não é programação)
    • Usa tags (etiquetas) para organizar o conteúdo
    • Compatível com navegadores modernos
    • Suporta semântica melhorada
    • Integração com CSS3 e JavaScript

    O HTML5 funciona através de um sistema de tags que indicam ao navegador como exibir o conteúdo.

    Exemplo básico:

    <!DOCTYPE html>
<html>
  <head>
    <title>Minha Página</title>
  </head>
  <body>
    <h1>Olá Mundo!</h1>
  </body>
</html>

    Como funciona:

    • O navegador lê o código HTML
    • Interpreta as tags
    • Renderiza (exibe) o conteúdo formatado
    • Aplica estilos do CSS se houver

    Toda página HTML5 deve seguir uma estrutura básica:

    Elementos principais:

    • <!DOCTYPE html> - Define que é um documento HTML5
    • <html> - Elemento raiz, contém toda a página
    • <head> - Cabeçalho, informações da página
    • <title> - Título que aparece na aba do navegador
    • <body> - Corpo, onde fica o conteúdo visível

    Meta tags importantes no <head>:

    • <meta charset="UTF-8"> - Codificação de caracteres
    • <meta name="viewport"> - Responsividade para mobile
    • <meta name="description"> - Descrição da página

    As tags de formatação definem como o texto será exibido:

    Tag Descrição Exemplo
    <h1> a <h6> Títulos (h1 maior, h6 menor) <h1>Título</h1>
    <p> Parágrafo <p>Texto aqui</p>
    <strong> Texto em negrito <strong>Negrito</strong>
    <em> Texto em itálico <em>Itálico</em>
    <br> Quebra de linha Linha 1<br>Linha 2

    Imagens:

    <img src="caminho/imagem.jpg" alt="Descrição">

    Vídeos:

    <video controls width="400">
  <source src="video.mp4" type="video/mp4">
  Seu navegador não suporta vídeos
</video>

    Áudio:

    <audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  Seu navegador não suporta áudio
</audio>

    Atributos importantes:

    • src - Caminho do arquivo
    • alt - Descrição alternativa (importante para acessibilidade)
    • controls - Adiciona controles de reprodução
    • width/height - Dimensões

    Listas não ordenadas:

    <ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

    Listas ordenadas:

    <ol>
  <li>Primeiro</li>
  <li>Segundo</li>
</ol>

    Tabelas:

    <table>
  <tr><th>Coluna 1</th><th>Coluna 2</th></tr>
  <tr><td>Dados 1</td><td>Dados 2</td></tr>
</table>

    Elementos de tabela:

    • <table> - Container da tabela
    • <tr> - Linha (table row)
    • <th> - Célula de cabeçalho
    • <td> - Célula de dados
